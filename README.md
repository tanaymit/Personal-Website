# Tanay Mittal — Portfolio Website

Personal portfolio website showcasing my experience, projects, and skills in Data Science, AI Engineering, and Machine Learning.

## Live Sections

- **Hero** — Introduction with profile photo, role titles, and social links
- **About Me** — Background, interests, and core competencies
- **Experience** — Timeline of roles at CMU, TresVista, and NUS
- **Projects** — Agentic Personal Finance Assistant, Diabetic Retinopathy Classification, Housing Sorted
- **Skills** — Programming, tools, data visualization, and AI/ML
- **Education** — Carnegie Mellon University and VIT Vellore
- **Contact** — Email, LinkedIn, Phone, and GitHub

## Tech Stack

- HTML5, CSS3, vanilla JavaScript
- [Inter](https://fonts.google.com/specimen/Inter) font (Google Fonts)
- [Font Awesome 6](https://fontawesome.com/) icons
- Fully responsive (mobile, tablet, desktop)

## Getting Started

1. Clone or download this repository
2. Open `index.html` in any browser

No build tools or dependencies required.
